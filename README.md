# PRACTICAS-9-10-EED-2026

Este repositorio contiene los distintos entregables del proyecto de Java:

## Entregable 1
La carpeta **EjemploEntornos** contiene el proyecto Java completo junto con sus tests realizados con JUnit.

## Entregable 2
La carpeta **informeDeCobertura** incluye el informe de cobertura del proyecto correspondiente al ejercicio 1.

## Entregable 3
El archivo **entregable3.pdf** contiene pantallazos del análisis realizado con SonarLint, junto con una breve explicación sobre qué son SonarLint y SonarQube.

## Entregable 4
La carpeta **EjemploEntornosRefactorizado** corresponde a la segunda versión del proyecto, ya refactorizado.

## Entregable 5
La carpeta **doc** contiene la documentación generada mediante JavaDoc.
